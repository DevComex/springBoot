活动规则

如何获取积分

游戏内活动积分：在活动期间，玩家在游戏内完成日常任务可以获得积分，日常活动项目包括：助人为乐、帮派日常挑战、副本系统、驱除邪灵、八仙梦境、探索世界、通天塔、悬赏任务、答题任务、师门任务。每种活动每天最多可获得2积分~

如何“点亮生日蛋糕”

玩家在本页面可进行许愿，召集小伙伴一起将生日蛋糕点亮后在生日蛋糕中抽奖，100%会获得奖励哦（获得奖池中的随机一个奖励）！每次许愿都会消耗积分，在越高层的生日蛋糕进行许愿消耗的积分就越多，奖励也更好哦，玩家每次许愿的内容不可以超过16个字哦，我们会尽快审核大家的愿望，通过审核的愿望将会显示在生日蛋糕内哟。

如何进行许愿抽奖

生日蛋糕只能依次被点亮，在该层未被完全点亮之前在该层的第一次许愿消耗积分较少，完全点亮后第一次在该层许愿就要消耗更多的积分啦。所有玩家在同层生日蛋糕进行多次许愿的时候，需要消耗的积分会越来越多哦。

如何获得奖励

玩家抽取到的称号、铭牌、银元宝奖励会直接发放到游戏中。其他奖励则需等到活动结束后统一发放~获得游戏内奖励的玩家可在收到系统邮箱提示后前往“南极仙翁”处领取。其他奖励会在活动结束后统一发放~其中实物奖励将发送至玩家所填写的地址中，宠物奖励将发至玩家参与活动绑定的角色中（称号效果是会覆盖的）。

温馨提示
玩家用鼠标停留在生日蛋糕上的时候就可以看见生日蛋糕中的奖励啦，点击生日蛋糕还可以看见大家许下的愿望哦~(部分奖池奖励种类较多，仅显示部分)玩家在游戏内获取的积分每小时会更新一次~

活动限制

本活动不对“内测专区”玩家开放；

玩家绑定角色后才可进行许愿；绑定角色不可更改~；

一个角色只能由一个帐号绑定参与活动，如若角色在交易前已绑定参与此活动则无法再次绑定该角色参与活动；

获得实物奖励的玩家需要认真填写自己的收件地址，如果因为地址填写问题导致奖励无法发放，则视为玩家放弃奖励~；
