活动规则

1召集令抽取活动时间：4月13日-4月25日
活动举办地点：浙江杭州
活动举办时间：4月29日-30日

2活动期间，玩家需在本页面登录账号，并绑定区组(该区组需有等级需≥20级的角色),即可参与活动；

3活动期间，玩家可通过【登陆并绑定区组】，每日获得3次抽奖机会。(判定区组数据读取可能会有30分钟延迟，请耐心等候)

4抽奖次数可累积，每日抽奖次数不限定；

5活动期间，每个光宇账号只可赢得一张邀请函；

6获得邀请函的玩家，为了确保您的获奖资格，请务必填写正确的个人信息（姓名、电话、邮箱、QQ、地址），工作人员将最迟于48小时以内与您联系；活动不收取任何费用，参与玩家的住宿以及餐饮将由官方统一负责，往返浙江杭州路费将最高报销500元；

7活动实物奖品将由《问道》官方统一安排联系派送，为了确保您的获奖资格，请务必填写正确的个人信息(姓名、电话、邮箱、QQ、地址)以免影响发奖；

8本次活动的解释权归光宇游戏所有。
